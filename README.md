# 🚀 Carreira 360 - Transformando o Futuro Profissional

O **Carreira 360** é uma plataforma inovadora projetada para conectar talentos, mentores e oportunidades. Focada no desenvolvimento profissional e na orientação de carreira, a plataforma oferece um ecossistema completo para estudantes (candidatos) e profissionais experientes (mentores).

## ✨ Funcionalidades Principais

### 🎓 Para Candidatos
- **Trilhas de Aprendizagem**: Percursos guiados com conteúdos pedagógicos, vídeos e materiais de apoio.
- **Comunidade & Fórum**: Espaço para debates, partilha de conhecimentos e networking.
- **Mentorias**: Agendamento de sessões individuais com profissionais de mercado.
- **Oportunidades**: Painel de vagas de emprego, estágios e bolsas de estudo.
- **Gamificação**: Sistema de XP, níveis e progressão para incentivar a aprendizagem.

### 🤝 Para Mentores
- **Dashboard Dedicado**: Gestão de pedidos de mentoria e disponibilidade.
- **Curadoria de Conteúdo**: Acesso privilegiado às trilhas para orientação de alunos.
- **Impacto Social**: Contribuição direta para a formação de novos talentos.
- **Interface RBAC**: Interface otimizada que separa as ferramentas de orientação das de aprendizagem (sem gamificação para mentores).

## 🛠️ Stack Tecnológica

- **Frontend**: React.js com Vite, TailwindCSS para estilização e Framer Motion para animações.
- **Backend**: Express.js API (TypeScript).
- **Base de Dados**: PostgreSQL com Drizzle ORM.
- **Segurança**: Autenticação via JWT e controle de acesso baseado em funções (RBAC).

## 🚀 Como Executar

### Pré-requisitos
- Node.js (v18 ou superior)
- pnpm

### Instalação
1. Clone o repositório
2. Instale as dependências:
   ```bash
   pnpm install
   ```
3. Configure o ficheiro `.env` com a sua `DATABASE_URL`.
4. Inicie o projeto:
   ```bash
   pnpm dev
   ```
